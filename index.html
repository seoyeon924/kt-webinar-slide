Tableau
Power BI

  • 완전한 인터랙티브 — hover · filter · drill · animation

Block 1 · BI 툴 vs Claude Code · 상세— — —
기존 BI 도구 — Tableau · Power BI
Tableau · Power BI가
적합한 경우
  • 전사 공유용 표준 리포트 — 비기술 사용자도 직관적으로 조회 가능한 안정적 리포팅 환경

  • 정해진 데이터 커넥터 (DB·DW·SaaS 40+종) — 대규모 운영 시스템 연동
  • 권한 관리 · 거버넌스 · 감사 로그 — 엔터프라이즈 보안 요건 충족

양이 아니라, 맥락이 결정한다.
적용 방법 → 팀의 분석 워크플로우를 agent.md 또는 CLAUDE.md로 작성. AI에게 "어떻게 생각해야 하는지"를 먼저 알려주기.

누가 하는가 vs 어떻게 하는가
—에이전트 — 역할·모델·권한이 고정된 AI 직원

—나누는 이유 — '누가'와 '어떻게'를 분리해 각각 독립적으로 수정 가능

샘플 데이터 생성 프롬프트
KT IT기술부문 클라우드 구독비용 모니터링 대시보드 제작용 샘플 데이터를 JavaScript 상수로 생성해줘. 서비스 15개 카테고리별로: AI도구: MS Copilot / AWS Bedrock / GCP Vertex AI 클라우드: AWS 인프라 / GCP 인프라 개발도구: GitHub Ent. / Jira+Confluence 협업: MS Teams / Slack / Zoom / Notion BI/디자인: Tableau / Power BI / Figma / Adobe CC 팀 6개: AI·데이터(45명) / 클라우드인프라(38명) / 개발플랫폼(52명) / 디지털혁신(35명) / 보안·운영(28명) / UX(22명) 기간: 18개월치 월별 · KT 규모 기준 (월 총비용 1~2억원) 각 서비스: base_cost / AI여부 / 팀별 사용 비중 포함 AI 서비스는 성장 트렌드 적용 · 이상 패턴 포함: - MS Copilot: 예산 103% 초과 - AWS Bedrock: 3개월 연속 15%+ 성장
EDA · 기초 분석 프롬프트
위에서 생성한 샘플 데이터(DATA 배열, 1,620 rows)를 기반으로 기초 탐색 분석을 수행해줘. 다음 항목을 분석하고 대시보드 설계에 활용할 요약을 작성해줘: 1. 데이터 규모 rows 수 · 컬럼 목록 · 기간 범위 2. 월별 총 비용 추세 최솟값 · 최댓값 · 평균 · 전체 성장률 3. 카테고리별 비용 비중 AI도구 / 클라우드 / 개발도구 / 협업 / BI·디자인 MoM 변화율 포함 4. 서비스별 예산 소진율 BUDGETS 대비 % — 초과 서비스 플래그 5. 팀별 지출 분포 TEAM_WEIGHTS 기준 상위 3팀 6. 이상 패턴 탐지 - 예산 초과 서비스 목록 - 3개월 연속 상승 서비스 - 활용률 60% 미만 서비스 (UTILIZATION 기준)
지표 설계 프롬프트
EDA 결과를 바탕으로 KT IT기술부문 클라우드 구독 모니터링 대시보드의 지표 체계를 설계해줘. 아래 형식으로 정의해줘: North Star 지표 1개 - 지표명 · 계산식 · 표시 형식(단위) - 비교 기준: 전월비 % · 예산 대비 % Supporting KPI 4개 - 지표명 · 계산식 · 분모 · 단위 - 각 KPI의 이상 임계값 기준 포함 대시보드 탭 구조 제안 - 탭 이름 · 각 탭의 주요 지표 목록 - 임원용 vs 실무자용 구분 포함 독자: IT기술부문 VP급 임원 + 팀 담당자 목적: 구독 비용 이상 탐지 + 계약 만료 관리
